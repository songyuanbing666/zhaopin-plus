# Zhaopin-Plus

智联招聘投递辅助脚本，帮助你高效管理求职投递流程。

## 功能特性

### 自动投递
- 批量自动投递智联招聘职位
- 支持投递间隔和上限设置
- 支持只收藏模式

### 岗位过滤
- 关键词过滤（黑名单/白名单模式）
- 屏蔽猎头和急聘岗位
- 薪资范围过滤

### 公司筛选
- 黑名单管理
- 公司成立时间过滤
- 注册资金过滤

### 沟通工具
- 自动回复已读未回消息
- 一键复制岗位信息

### 数据统计
- 7天投递统计图表

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 插件
2. 点击「创建新脚本」
3. 复制 zhaopin-plus.user.js 内容粘贴进去
4. 保存并打开智联招聘网站

## 使用说明

1. 打开智联招聘职位搜索页面
2. 点击左下角蓝色按钮开始自动投递
3. 通过设置按钮配置各项参数

## 注意事项

- 请合理控制投递频率，避免账号风控
- 建议投递间隔设置在 3-10 秒之间
- 本脚本仅供学习交流使用
